# Velocity

**Velocity is a real-time analytics dashboard** that gives your team a live view
of traffic, conversions, and revenue as they happen. Plug in your data source and
watch the numbers update second by second.

## Features

- 📊 Live, real-time charts that stream updates over WebSockets
- 🚦 Conversion funnels and revenue tracking out of the box
- 🔔 Threshold alerts when a metric crosses a limit
- 👥 Multi-tenant workspaces for your whole team

## Getting started

```bash
npm install
npm start
```

Then open <http://localhost:8080> to view your dashboard.
